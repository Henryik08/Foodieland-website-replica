# Foodieland Website Replica

A beautiful and responsive food website replica built with HTML and CSS. This project showcases a modern food-focused website with multiple pages for browsing recipes, reading food blogs, and getting in touch.

## Features

- **Home Page** - Main landing page with featured content
- **Blog Page** - Food blog with articles and recipes
- **Contact Page** - Contact form for user inquiries
- **Responsive Design** - Clean and modern UI
- **Static Pages** - Pure HTML and CSS (no backend required)

## Project Structure

```
├── index.html           # Home page
├── index.css            # Home page styles
├── blogPage/
│   ├── blog.html        # Blog page
│   └── blog.css         # Blog page styles
├── contactPage/
│   ├── contact.html     # Contact page
│   └── contact.css      # Contact page styles
├── images/              # Image assets
└── README.md            # Project documentation
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and layout

## Pages

- **Home** - Main landing page introducing Foodieland
- **Blog** - Articles and recipes about food
- **Contact** - Get in touch with us
